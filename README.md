# Snake Game (Змейка)

Классическая игра "Змейка" на Windows Forms (C#).

## Управление
- Стрелки (↑ ↓ ← →) — управление движением змейки

## Особенности
- С увеличением счёта игра ускоряется
- Отслеживается уровень (каждые 5 очков)
- При столкновении со стеной или своим телом игра завершается с предложением начать заново

## Как запустить
1. Установите .NET 8.0 SDK
2. Откройте решение в Visual Studio (или другом редакторе)
3. Нажмите F5 для запуска
